## 有什么用？
无需翻找令人疑惑的配置文件,直接在UI上编辑  
方便快捷定时备份MC服务端(还有你的各种资源文件),关闭后再备份有效避免了文件被锁定的问题  
快捷地查看/管理多个服务端（同一窗口上）  
快速地浏览/下载/安装插件模组  
快速编辑服务端配置文件  
## 初步配置
若使用RCON：  
在服务端根目录下找到server.properties  
修改enable-rcon=true  
修改rcon.password=你喜欢的密码  
修改rcon.port=你喜欢的端口（其实默认即可）  
在BBZBackup程序中的RCON(cli)设置窗口填写你刚刚设置的密码和端口并保存  
配置好所有可配置的（除非你不用）即可
