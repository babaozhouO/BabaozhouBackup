## ❓ 产品价值
- **简化配置**：可视化编辑代替手动修改配置文件

## ⚙️ 初始化配置 (RCON 模式)
1. **服务端配置**  
   修改 `server.properties` 文件：
   ```properties
   enable-rcon=true
   rcon.password=你的密码
   rcon.port=25575  # 建议保持默认
   ```
